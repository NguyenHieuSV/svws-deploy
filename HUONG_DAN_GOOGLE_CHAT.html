Cách A — Nhắn riêng từng người

Dành cho người có quyền quản trị Google Workspace của công ty.

  1. Tạo project trên Google Cloud

    Mở console.cloud.google.com rồi tạo project mới, đặt tên SVWS Bot.

  2. Bật Google Chat API

    Trong project vừa tạo, vào APIs & Services › Library, tìm Google Chat API rồi bấm Enable.

  3. Tạo Service Account và tải khóa

    • Vào APIs & Services › Credentials › Create Credentials › Service account
    • Đặt tên svws-nhac-viec, bấm Create and continue rồi Done
    • Bấm vào service account vừa tạo, sang tab Keys › Add key › Create new key
    • Chọn định dạng JSON › Create. Máy sẽ tải về một file .json

    Giữ file này cẩn thận. Nó là chìa khóa gửi tin — ai có file cũng nhắn được dưới danh nghĩa công ty. Không gửi qua chat công khai, không commit lên GitHub.

  4. Cấu hình Chat app

    Vào Google Chat API › Configuration và điền:

    • App name: Nhắc việc SVWS
    • Description: Nhắc việc tự động từ hệ thống SVWS
    • Avatar URL: để trống, hoặc dùng logo công ty
    • Functionality: tích Receive 1:1 messages
    • Connection settings: chọn App URL và để trống — bot chỉ gửi, không nhận
    • Visibility: cho phép trong miền watersolutions.company

    Bấm Save.

  5. Không được bỏ qua

    Cài Chat app cho toàn công ty

    Thiếu bước này thì bot không nhắn riêng cho bất kỳ ai được, dù bốn bước trên đã làm đúng hết.

    • Mở admin.google.com
    • Vào Apps › Google Workspace › Google Chat › Manage Chat apps
    • Tìm app Nhắc việc SVWS và đặt trạng thái Allowed / ON for everyone

  6. Khai báo vào hệ thống SVWS

    Mở dashboard.render.com › dịch vụ svws-app › tab Environment, thêm hai biến:

    Biến môi trường cần thêm
    CHAT_PROVIDER
    APP
    GCHAT_SERVICE_ACCOUNT
    Dán nguyên văn toàn bộ nội dung file .json ở bước 3

    Bấm Save changes. Render tự khởi động lại, khoảng 2 phút.

    Khi dán file JSON: dán nguyên văn, không thêm dấu xuống dòng, không xóa bớt ký tự nào. Hệ thống có tự sửa trường hợp private_key bị mất xuống dòng, nhưng dán đúng ngay từ đầu vẫn chắc ăn hơn.

Cách B — Gửi vào một Phòng chung

Không cần quyền quản trị. Tin nhắc đăng vào một Phòng chat, mọi người trong phòng cùng thấy.

  1. Tạo Phòng chat

    Mở Google Chat, tạo Space mới tên Nhắc việc SVWS, rồi mời nhân viên vào phòng.

  2. Tạo webhook cho phòng

    Trong phòng đó, bấm vào tên phòng › Apps & integrations › Webhooks › Add webhook. Đặt tên SVWS, bấm Save.

  3. Sao chép đường link webhook

    Google sẽ hiện một đường link dài. Sao chép toàn bộ link đó.

  4. Khai báo vào hệ thống SVWS

    Mở dashboard.render.com › svws-app › Environment, thêm hai biến rồi bấm Save changes:

    Biến môi trường cần thêm
    CHAT_PROVIDER
    WEBHOOK
    GCHAT_WEBHOOK_URL
    Đường link vừa sao chép

Sau này làm xong Cách A, chỉ cần đổi CHAT_PROVIDER thành APP và thêm GCHAT_SERVICE_ACCOUNT. Không phải sửa phần mềm.

Kiểm tra sau khi cài đặt

Làm theo thứ tự này để biết chắc đã chạy thật.

  • Đăng nhập SVWS bằng tài khoản CEO hoặc Admin
  • Vào Working time & Report › tab Work Reminder
  • Kéo xuống panel ⚙️ Kết nối Google Chat — chế độ phải đã đổi khỏi DEMO
  • Bấm 📨 Gửi tin thử cho tôi, rồi mở Google Chat kiểm tra
  • Bấm 📰 Xem thử bản tin đầu ngày để xem trước nội dung bản tin tổng hợp

Điều kiện bắt buộc để nhắn riêng: email nhân viên trong SVWS phải trùng email Google Workspace. Kiểm tra tại Nhân sự & Lương › Hồ sơ lương › cột Email.
